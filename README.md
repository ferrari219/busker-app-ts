구축해보면서 더 나은 운영/관리 방법에 대한 고민이 많았습니다.
오픈 후에도 계속 깔끔한 코드로 중복 제작을 방지하고 누가 처음하더라도 한사람이 작성한것처럼 운영하려면 지금의 방식으로 하면 어떨까 하는 생각에서 고민하다 제작했습니다. 스토리북 구성으로 디자이너나 기획자와의 협업에도 큰 도움이 되더군요.
